# 💳 Bankist App

This is a simple, responsive banking application built with JavaScript.  
It simulates basic banking operations like logging in, transferring money, requesting loans, and viewing account activity. This project focused on improving my understanding of arrays, dates, and DOM manipulation.

## Features
- User login system  
- Money transfers between accounts  
- Loan request functionality  
- Transaction history display  
- Date and currency formatting  
- Responsive design  

## Built With
- HTML  
- CSS  
- JavaScript
